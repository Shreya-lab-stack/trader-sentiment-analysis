# Trader Behavior Analysis

This project analyzes how market sentiment (Fear & Greed Index) influences trader profitability, win rate, and risk behavior.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Google Colab

## Key Insights
- Highest average profits occur during Greed periods
- Win rates peak during Extreme Greed
- Neutral sentiment leads to weakest trader performance
- Most trades close at breakeven, indicating skewed profit distribution

## Files
- trader_sentiment_analysis.ipynb – analysis notebook
- ds_report.pdf – detailed project report
- *.png – visualizations
